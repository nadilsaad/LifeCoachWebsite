# Life Coach Web Application

Life Coach is a web-based personal development application designed to help users build positive habits, track progress, and maintain consistency in their self-improvement journey. The platform provides users with a supportive digital environment for habit management, journaling, and personal growth.

## Features

- User registration and login
- Habit tracking
- Daily progress monitoring
- Personal journaling
- Interactive dashboard
- Clean and user-friendly interface

## Project Purpose

The purpose of this project is to help users improve their daily lives by building discipline, maintaining positive routines, and supporting personal development through a structured digital platform.
